# ChatRD Updates

## 🫡 General

- CSS Code Revamp.
  - This was inevitable since most of the CSS visuals are being inherited from the ChatRD "Default Mode", so I made the modes separate from each other. Due to the amount of requests, it was being difficult to maintain properly without glitching other parts. Nothing changes on the front-end, **but if you customized ChatRD using Custom CSS, there's a chance that you might experience visual glitches.** (To be honest, I think the alignment and animations are better and smoother).

- Removed "Multistreamer Mode".
  - Twitch was the only platform enforcing penalties against multi chat widgets/overlays. Now they removed this from their Terms of Service.
- Removed "Platform Dots".
  - Same reason as the above.

- Reworked the display animation for chats and events.
- Reordered options and switches.
- Fade In/Fade Out animation added when you click the **Platform Selection** icon on the Chat Module.

## YouTube

- Changed the Moderator icon from Wrench to Shield.
- The badges for Sub, Mod, Streamer and Verified are now circular.
- The mysterious ":awesome:" emote, which were not showing before due to how the YouTube Payload is, is now forcibly shown. 😁
